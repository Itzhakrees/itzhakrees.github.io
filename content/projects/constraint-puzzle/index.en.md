# Constraint-Based Puzzle Prototype

## Overview

This game jam prototype uses a single rule constraint to create compact teaching, short-term planning, and a readable difficulty ramp.

{{ video bilibili:BV1xx411c7mD title="Constraint puzzle prototype capture" }}

## Design Ownership

- Converted the jam theme into one reusable rule.
- Designed early puzzles that teach through safe failure.
- Added late puzzles that combine timing, space, and resource pressure.

## Lessons

The prototype worked best when each puzzle introduced one new relationship. When a room tried to teach timing and spatial planning at the same time, players solved it more slowly but remembered less.

## Next Steps

- Reduce the number of tutorial prompts.
- Add clearer end-state feedback.
- Test whether the rule still works when puzzle rooms become larger.
